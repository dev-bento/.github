# Dev Bento 🍱

Tiny local CLIs for AI-assisted indie developers.

Don’t dump your repo. Pack it into a bento.

## What is Dev Bento?

Dev Bento is a collection of small local CLI tools that prepare your project context for AI.

Each tool scans or summarizes one part of your project and generates a clean Markdown report you can hand to ChatGPT, Claude, Codex, Cursor, or Gemini.

## Tools

Coming soon:

- **Secret Bento** — scan your repo locally for leaked secrets and generate an AI-ready remediation report.
- **Repo Bento** — turn your repository into a clean Markdown brief for AI maintenance.
- **Git Bento** — explain your Git state in plain English and generate an AI handoff.
- **Env Bento** — compare environment variable usage and examples.
- **Launch Bento** — prepare a side project for public launch.

## Philosophy

Local first.  
Markdown first.  
No magic. No lock-in. No code upload.

We pack the context.  
You and your AI cook from there.
